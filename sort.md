## 选择
选择排序的逻辑很简单，选一个**最小**的放到第一个，次小的放到第二个，以此类推...    
就如同我们平时整理东西一样，你看桌子上一堆书，怎么放整齐呢？你把最小的放上面，再把次小的放第二层以此类推。
```C
void selectSort(datatype *p,int len){
	int i,j,now;
	for(i=0;i<len;i++){
		now = i;
		for(j=i+1;j<len;j++){
		    if(p[now] >p[j]){
				now = j;
			}
		}
        if(now != i){
            swap(p+i,p+now);
        }
	}
}
```
